🏥 Intelligent Pharmacology Medicine Import System

✨ Key Features

🔤 Field Name Auto-Correction

Automatically maps different field names to the correct schema:

  • medicine → Medicine Name
  • drug_class → Drug Class
  • uses → Clinical Uses
  • side_effects → Adverse Effects

📝 Clinical Use Processing

Converts text descriptions into structured format:

Input: "Treats bacterial infections including UTI (dysuria, urgency, frequency)"
Output: Structured clinical use with extracted symptoms

🔍 Symptom Extraction

Automatically extracts symptoms from natural language:

  • Clinical uses text
  • Condition descriptions
  • Common medical terminology

🛡️ Safety Rule Normalization

Converts various safety rule formats:

"avoid_in_pregnancy": "true" → avoidInPregnancy: true
"neonatal_warning": "avoid" → avoidInNeonates: true
"caution_in_breastfeeding": "yes" → cautionInBreastfeeding: true

📋 Missing Field Handling

Gracefully handles missing data:

  • Sets appropriate defaults for optional fields
  • Provides warnings for missing critical fields
  • Never fails import due to missing optional data

🧪 Sample Import Data

Try importing the sample file sample-intelligent-import.json to see the intelligent auto-correction in action.

Example Input (Mixed Formats):

{ "medicine": "Paracetamol", "drug_class": "Analgesic", "mechanism": "Blocks pain signals", "uses": "Fever, headache, muscle pain", "side_effects": "Nausea, stomach upset", "avoid_in_pregnancy": "false" }

Auto-Correction Results:

  • ✓ medicine → name
  • ✓ drug_class → drugClass
  • ✓ mechanism → mechanismOfAction
  • ✓ uses → clinicalUses
  • ✓ side_effects → adverseEffects
  • ✓ "false" → false (boolean)

📊 Import Process Flow

  1. Field Normalization - Map field names to standard schema
  2. Clinical Use Processing - Convert text to structured format
  3. Symptom Extraction - Extract symptoms from text
  4. Safety Rule Normalization - Convert safety rules to standard format
  5. Validation - Check for critical missing fields
  6. Summary Generation - Create detailed import report
